Adding multi column / grouped headers

However the data is set, it is possible to add grouping to headers by specifying a groups array for the header columns 
{
  columns:[ // object for each column
    { value:"Name", groups:["Group 1"]},
    { value:"Age" },
    { value:"Gender", groups:["Group 2"]},
    { value:"Date of Birth", groups:["Group 2"]},
    { value:"Salary (£)", groups:["Group 2"]}
  ]
}

Wrapping header text for individual columns

It is possible to override the allowHeaderWrap option for an individual column by providing the allowWrap property in the header colums: 
{
  head: {
    columns:[ // object for each column
      { value:"Name" },
      { value:"Age" },
      { value:"Gender" },
      { value:"Date of Birth", allowWrap: true}, // Column will wrap if needed
      { value:"Salary (£)", allowWrap: false } // Column will never wrap
    ]
  }
}

Making rows unselectable

It is possible to prevent rows from being selected by providing the unselectable property in the row data. This will cause the row tick to be disabled and prevent the row from being selected by any means. 
{
  head:...
  body: [
    {
      unselectable: true,
      columns: [
        ...// Row data
      ]
    }
  ]
}
If this value is set, the row will always be unselectable and as such will not be passed through the rowSelectableLookup function.
