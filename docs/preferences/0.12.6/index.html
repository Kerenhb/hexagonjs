Preferences
A method for storing user settings across applications for things like time zone and locale settings.
Examples

HTML

<button id="pref" class="hx-btn hx-positive">Preferences <i class="fa fa-cog"></i></button>

JavaScript

hx.preferences.supportedLocales = ['en-gb', 'de', 'en-us', 'be']
hx.select('#pref').on('click', function(){
  hx.preferences.show()
})

Prototypes
UserPreferencesextendsEventEmitter
A class for getting and setting user preferences for the current domain.
Properties
supportedLocalesArray
An array of locale codes that are supported by your application. The list must be a lower case list of locale codes: 
[
  'en-gb',
  'en',
  'de',
  'be'
]
Methods
clearPreferences
Clears the stored user's preferences - after calling this, the current users settings will be reset to the default values
getLocaleListArray[String]
Returns

The list of all the locales listed by Moment.js

callbackerrorObjectdataObject
The function to call when the preferences have been retrieved.
Arguments
error
The error returned when getting the data. Passed as 'undefined' when no error has occured.
data
The data object returned from the user's local storage. This will have the same properties as the preferences property.
Properties
localeString
The user's selected locale.
timeZoneString
The user's selected time zone.
The time zone is intended to be used with Moment.js Timezone and the full list of timezones is retrieved using moment.tz.names() The value is stored as a string and can be any value from the moment.tz list.
